HTML Tag Description
 <!DOCTYPE html>  This document is HTML5 document, It must only appear once, at the top of the page
 <head>  This root element or beginning of HTML code that is divided into two parts (1) head and (2) body. The visible or display part is body
 <meta>  metainformation about the page. Meta means data about data, that means it will include all the information about and required by the page
 <em> for making an emphasis on text by italicizing it
 <body>  Visibile container of the HTML page
<i>
 Italic the text it is used inside some tags, after the tag ends there is no new line
 <strong>  same as bold but it tells the emphasis on text to the browser. I consider it to be semantic based programming
 <img> image tag that is used to display an image, it has elements such as src for actual location of image, alt for alternative text if image is not displayed, width and hieght are properites of image show, Images tag doesnt go to new line
 <marquee>  for having a moving text
 <table>  Creates a table that has rows (<tr>) and data (<td>)
